# 🏥 AI Medical Surgery Planner

An AI-powered surgical planning system that converts CT/MRI scans into interactive 3D visualizations.

## 🚀 Features (Planned)

- CT/MRI Upload
- DICOM & NIfTI Support
- AI Organ Segmentation
- Tumor Detection
- Blood Vessel Segmentation
- Interactive 3D Viewer
- AI-powered Surgery Explanation
- PDF Report Generation

## 🛠 Tech Stack

- Python
- FastAPI
- React
- PyTorch
- MONAI
- Three.js
- VTK.js
- NumPy
- OpenCV
